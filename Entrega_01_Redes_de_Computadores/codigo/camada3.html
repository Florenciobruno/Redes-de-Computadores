Camada 03 (Rede)

A camada de rede é composta por:

  • Endereçamento
  • Encapsulamento
  • Desencapsulamento
  • Roteamento

Endereçamento

Fragmentos individuais de dados precisam ser direcionados a um dispositivo final, este dispositivo precisa ter um endereço único

Este endereço ficou conhecido como endereço IP

Ou endereço lógico em contraste com o endereço físico da camada de enlace

Encapsulamento

Os fragmentos individuais também devem conter estes endereços.

Ao fazer referência à camada de rede,chamamos de pacote.

Quando se cria um pacote algumas informaçoes são importantes como, o endereço do host, esse enderço é chamado de destino

Desencapsulamento

O pacote chega ao host de destino e é processado na camada 03.

O host examina o endereço de destino e verifica se o pacote estava endereçado para ele mesmo, se o pacote estiver correto ele é

desencapsulado pela camada de Rede.

Roteamento

A camada de rede precisa fornecer meios para que os pacotes cheguem ao seu host de destino.

Os hosts de origem e destino nem sempre estão conectados na mesma rede.

Os roteadores são dispositivos intermediarios que conectam as redes.

Este processo é conheciso como roteamento.
